# **LUCID DRIFT**

### *Where Memory Wakes in Fragments*

---

## ✦ Overview

*LUCID DRIFT* is an experimental system exploring memory as something that is not fixed, but **fragmented, unstable, and generative**.

Rather than preserving memory, the project **reconstructs it continuously** — through sound, visuals, and interaction.

This repository contains the **conceptual foundation** of the work.

It exists as one layer within a larger system:

* Visual Design System
* Interactive Prototype
* Generative AI Integration
* Spatial Installation

→ This is not the final work.
→ This is the **logic that allows the work to exist.**

# **2 · Artist & Project Framing**

Hari Tris is a multidisciplinary artist whose work moves across **art direction, sound, and moving image**.

Her early memories are shaped by her father’s midnight composing sessions —
Vietnamese love songs that existed first as **feeling, not language**.

*LUCID DRIFT* explores:

* memory
* fragmentation
* reconstruction

within a broader research framework:

## → *Orphaned Archives*

# **3 · Thematic Frame — Orphaned Archives**

> When a person dies, who owns their data?

An **orphaned archive** is memory that remains without:

* ownership
* authorship
* system of care

The project investigates tension between:

**Analog Memory**

* cassette tapes
* handwritten scores
* physical traces

**Digital Systems**

* compression
* storage
* obsolescence

→ Memory is preserved
→ but also flattened, extracted, consumed

---

## ✦ Core Question

> Can memory exist outside extractive systems?
