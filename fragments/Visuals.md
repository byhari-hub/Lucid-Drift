# Fragments

This folder contains raw visual and audiovisual materials used in *LUCID DRIFT*.

These are not final assets.

They are **modular memory units** — fragments that can be:
- recombined
- transformed
- reinterpreted

---

## Structure

Fragments are organized by **emotional and spatial motifs**, not chronology.

Each file represents:
→ a partial memory  
→ an unstable visual trace  

---

## Categories

- /clouds/
- /rivers/
- /thresholds/
- /urban/
- /youth/
- /noise/

---

## Usage

Fragments are used for:
- AI transformation
- real-time interaction
- visual recomposition
- system testing

No fragment is fixed in meaning.

Each interaction produces a new interpretation.
