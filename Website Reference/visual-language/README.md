### Fragmented Presence

The visual system resists resolution.  
No image is complete, stable, or fully accessible.

What appears is always partial — emerging, dissolving, or withheld.

---

### Characteristics

- High-contrast monochrome palette (black, off-white, grey)  
- Overexposed or luminous imagery — detail lost through intensity  
- Grain, noise, and compression artifacts as inherent texture  
- Hybrid visual sources:
  - archival traces (cassette, handwritten scores)
  - field recordings (subjective, time-based, imperfect)
  - AI-generated hallucinations (non-real, unstable states)

---

### Behavior

- Images load progressively, partially, or with delay  
- Cropping feels unstable — frames appear cut, misaligned, or incomplete  
- Layers overlap and drift, never locking into a fixed composition  
- Visual states remain unresolved — no final or “perfect” frame  

---

### Conceptual Mapping

- Orphaned archive → incomplete, dislocated frames  
- Memory decay → noise, blur, and visual degradation  
- Recomposition → layering, collision, and drift  
- Absence → void, silence, and sustained negative space
